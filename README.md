## Caesar Cipher Belarusian Language (CCBL)
CCBL is a utility on Python for encoding and decoding Caesar cipher for Belarusian language
### To use it, you need:
- Python (seriously, that's all)
#### To change alphabet, you need to:
- run CCBL.py and paste your alphabet in "Custom Alphabet" box
##### Examples of alphabets:
- ENG: abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
- RUS: абвгдеёжзийклмнопрстуфхцчшщъыьэюяАБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ
- BEL: абвгдеёжзійклмнопрстуўфхцчшыьэюяАБВГДЕЁЖЗІЙКЛМНОПРСТУЎФХЦЧШЫЬЭЮЯ
